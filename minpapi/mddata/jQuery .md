jQuery 是一个 JavaScript 库。
jQuery 极大地简化了 JavaScript 编程。
jQuery 很容易学习。

jQuery 语法实例
$(this).hide()
演示 jQuery 的 hide() 函数，隐藏当前的 HTML 元素。
$("p").hide()
演示 jQuery 的 hide() 函数，隐藏所有 <p> 元素。
$(".test").hide()
演示 jQuery 的 hide() 函数，隐藏所有 class="test" 的元素。
$("#test").hide()
演示 jQuery 的 hide() 函数，隐藏 id="test" 的元素。
